Brief Description: Two-pass connected component labeling (CCL) is a classic algorithm to detect and label distinct connected regions (components) in a binary image. The output is a labeled image where each connected region has a unique label ID. To achieve this, the first pass is to assign temporal labels and record label equivalences; the second pass is to resolve equivalence relations and do the re-label.
